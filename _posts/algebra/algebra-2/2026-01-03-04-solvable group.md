# 可解群

## composition series

::: definition
$G$正规列定义为一列有限正规子群形如： $$G=H_0\unrhd H_1 \unrhd H_2 \unrhd \cdots \unrhd H_m = \{1\}$$
:::

::: definition
给定两个正规列 $$G=H_0\unrhd H_1 \unrhd H_2 \unrhd \cdots \unrhd H_m = {1}$$ $$G=K_0\unrhd K_1 \unrhd K_2 \unrhd \cdots \unrhd K_n = {1}$$ 称$\{H_i\}$是$\{K_i\}$的refinement (as multiset)如果$\{H_i\}\supset\{K_i\}$。
:::

::: definition
如果$H_i/H_{i+1}$是单群,称$(H_i)$为composition series。
:::

::: proposition
:::

::: theorem
给定两个正规列$(H_i),(K_j)$，则存在$(H_i)\&(K_j)$的加细等价。
:::

::: theorem
如果$G$包含composition series，那么这些composition series 都等价。
:::
