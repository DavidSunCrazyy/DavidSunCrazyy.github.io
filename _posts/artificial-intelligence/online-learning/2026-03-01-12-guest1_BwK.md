本文研究了**带背包约束的老虎机问题（Bandits with Knapsacks, BwK）**，这是经典多臂老虎机模型在资源约束下的重要扩展。在每个时间步，玩家选择一个臂不仅获得随机奖励，还会消耗多种资源的随机成本，总消耗受限于给定的初始预算。文章首先建立了静态 BwK 的线性规划基准，并通过拉格朗日对偶方法将其转化为一个博弈问题：玩家最大化「奖励 − 价格加权成本」，而对偶方（资源定价）最小化总成本。基于此框架，文章提出了一种结合 UCB（奖励上置信界）和 LCB（成本下置信界）的在线拉格朗日算法：玩家通过在线镜像下降（OMD）交替更新动作分布和资源价格。分析表明，在适当步长选择下，算法可实现 $\widetilde{O}(\sqrt{T})$ 的遗憾上界。文章中间穿插介绍了**在线凸优化**的核心工具——在线镜像下降和乘法权重更新（MWU），并给出了标准的遗憾分解证明，为理解 BwK 及其他约束在线学习问题提供了坚实的理论基础。

# 带背包的老虎机问题（BwK）

##### 1. 静态 BwK

- 时间范围：$T$ 臂的数量：$K$ 资源的数量：$d$

- 每个臂 $i$ 对应奖励和成本 $(r_i, C_i)$，其中 $r_i \in [0,1]$，$C_i \in [0,1]^d$。

- 在每个时间步 $t$，玩家选择一个臂进行操作，获得奖励 $r_{t,i}$ 并消耗成本 $C_{t,i}$。（$r_{t,i} \sim N(r_i, 1)$，$C_{t,i} \sim N(C_i, I_d)$）。

- 问题：给定总预算 $B I_d$，如何最大化累积奖励？

##### 2. 基准：

$\text{OPT} := \max \sum_{t=1}^T \sum_{i=1}^K r_i x_{t,i}$ $\text{满足 } (x_{t,i})_{i=1}^K \in \Delta_K \quad \forall t \in [T],$ $\sum_{t=1}^T \sum_{i=1}^K x_{t,i} C_i \leq B I_d.$

**命题 1.** $\text{OPT} = \max_{x \in \Delta_K} T \langle r, x \rangle \quad \text{其中 } r = (r_i)_{i=1}^K \in \mathbb{R}^K$ $\text{满足 } C x \leq \frac{B}{T} I_d, \quad \text{其中 } C = (C_i)_{i=1}^K \in \mathbb{R}^{d \times K}$

\[证明思路：詹森不等式。\]

##### 3. 直观理解与算法：

考虑以下优化问题的拉格朗日函数：$\max_{x \in \Delta_K} \langle r, x \rangle \quad \text{(*)}$ $\text{满足 } C x \leq b I_d, \quad \text{其中 } b = \frac{B}{T}.$

$\mathcal{L}(x,y) = \langle r, x \rangle + \langle y, b I_d - C x \rangle$ $\text{OPT} = \min_{y \in \mathbb{R}_+^d} \max_{x \in \Delta_K} \mathcal{L}(x,y) \implies \text{根据此规则迭代更新 } x, y.$

- **算法：**

    - 初始化 $y_1 = \frac{1}{d} \mathbf{1}_d$，$\eta = \frac{1}{b}$，$\varepsilon = \sqrt{T}$。

    - 对于 $t = 1, \dots$ 执行：

        - 估计 $r_t \leftarrow \text{UCB}(r)$，$C_t \leftarrow \text{LCB}(C)$

        - $x_t = \arg\max_{x \in \Delta_K} \langle r_t - \eta C_t^\top y_t, x \rangle$，按 $x_t$ 选择臂。

        - $y_{t+1} = \arg\min_{y \in \Delta_d} -\langle y, C_t x_t \rangle + \frac{1}{\varepsilon} D_{KL}(y, y_t)$

##### 4. 重要旁述：OMD、MWU。

**在线凸优化**：$x \in \mathbb{R}^d$ 为凸集。

每个时间步 $t$，玩家 1 选择 $x_t \in X$，

玩家 2 选择一个凸损失函数 $f_t: X \to \mathbb{R}$。

目标：$\min \sum_{t=1}^T (f_t(x_t) - f_t(x^*))$，其中 $x^* = \arg\min_{x \in X} \sum_{t=1}^T f_t(x)$。

令 $g_t \in \partial f_t(x_t)$ 为 $f_t$ 在 $x_t$ 处的次梯度。

**在线镜像下降**：$x_{t+1} = \arg\min_{x \in X} \langle g_t, x \rangle + \frac{1}{\varepsilon} D_{KL}(x, x_t)$。

**乘法权重更新**：$x_{t+1, i} = \frac{x_{t,i} \exp(-\varepsilon g_{t,i})}{\sum_{j=1}^d x_{t,j} \exp(-\varepsilon g_{t,j})}$。（$X = \Delta_d$）

**定理。** $\sum_{t=1}^T \left[ f_t(x_t) - f_t(x^*) \right] \leq \frac{1}{\varepsilon} D_{KL}(x^*, x_1) + \frac{\varepsilon}{2} \sum_{t=1}^T \|g_t\|_\infty^2.$ 特别地，取 $\varepsilon = \frac{1}{\sqrt{T}}$，右侧为 $O(\sqrt{T})$。

**证明：** $f_t(x_t) - f_t(x^*) \leq \langle g_t, x_t - x^* \rangle = \langle g_t, x_t - x_{t+1} \rangle + \langle g_t, x_{t+1} - x^* \rangle$ $\langle g_t, x_t - x_{t+1} \rangle \leq \|g_t\|_\infty \|x_t - x_{t+1}\|_1 \leq \frac{\varepsilon}{2} \|g_t\|_\infty^2 + \frac{1}{2\varepsilon} \|x_t - x_{t+1}\|_1^2$ $\leq \frac{\varepsilon}{2} \|g_t\|_\infty^2 + \frac{1}{\varepsilon} D_{KL}(x_{t+1}, x_t) \quad \text{(Pinsker)}$ 注意到，$D_{KL}(x_{t+1}, x_t) = D_{KL}(x^*, x_t) - D(x^*, x_{t+1}) + \varepsilon \langle g_t, x^* - x_{t+1} \rangle$。$\Rightarrow f_t(x_t) - f_t(x^*) \leq \frac{1}{\varepsilon} \left[ D_{KL}(x^*, x_t) - D_{KL}(x^*, x_{t+1}) \right] + \frac{\varepsilon}{2} \|g_t\|_\infty^2.$ 对 $t$ 求和。$\square$

##### 5. 分析：

令 $\tau$ 表示停止时间，$x^*$ 为最优策略。

###### 1° \[高层思路\]

$\text{REW} = \mathbb{E} \left[ \sum_{t=1}^T \langle r, x_t \rangle \right] \geq \mathbb{E} \left[ \sum_{t=1}^T \langle r_t, x_t \rangle \right] - \widetilde{O}(\sqrt{T}) \quad \text{(UCB)} \quad \text{[1]}$ $= \mathbb{E} \left[ \sum_{t=1}^T \langle r_t - \eta C_t^\top y_t, x_t \rangle \right] + \eta \mathbb{E} \left[ \sum_{t=1}^T \langle y_t, C_t x_t \rangle \right] - \widetilde{O}(\sqrt{T})$ $\geq \underbrace{\mathbb{E} \left[ \sum_{t=1}^T \langle r_t - \eta C_t^\top y_t, x^* \rangle \right]}_{\text{(I)}} + \eta \underbrace{\mathbb{E} \left[ \max_{y \in \Delta_d} \sum_{t=1}^T \langle y, C_t x_t \rangle \right]}_{\text{(II)}} - \widetilde{O}(\sqrt{T} + \eta \sqrt{T})$

$\text{(I)} \geq \mathbb{E} \left[ \sum_{t=1}^T \langle r - \eta C^\top y_t, x^* \rangle \right] - O(1) \quad \text{(LCB)}$ $= \mathbb{E}[\tau \langle r, x^* \rangle] - \mathbb{E} \left[ \eta \langle \sum_{t=1}^T y_t, C x^* \rangle \right] - O(1)$ $\geq \mathbb{E}[\tau \langle r, x^* \rangle] - \mathbb{E} \left[ \eta \langle \sum_{t=1}^T y_t, b I_d \rangle \right] - O(1) = \mathbb{E}[\tau] (\langle r, x^* \rangle - b \eta) - O(1)$

根据 $\tau$ 的定义，我们有 $\mathbb{E} \left[ \max_{y \in \Delta_d} \sum_{t=1}^T \langle y, C x_t \rangle \right] \geq T b$。

$\text{(II)} \geq \eta T b - \widetilde{O}(\eta \sqrt{T}). \quad \text{(LCB)} \quad \text{[2]}$

取 $\eta = \frac{1}{b}$，得证。$\square$

###### 2° 误差分析

**\[1\]:** 固定 $i \in [K]$，定义 $N_t(i) = \sum_{s=1}^t \mathbf{1}_{\{x_{s,i}=1\}}$。

则事件 $E_{t,i}^r = \left\{ 0 \leq r_{t,i} - r_i \leq \widetilde{O}\left( \frac{1}{\sqrt{N_t(i)}} \right) \right\}$ 以概率 $\geq 1 - \frac{1}{K T^2}$ 成立。

$\Rightarrow E^r = \bigcap_{t,i} E_{t,i}^r$ 以概率 $\geq 1 - \frac{1}{T}$ 成立。

在此事件下，$\mathbb{E} \left[ \sum_{t=1}^T (r_{t,i} - r_i) x_{t,i} \mathbf{1}_{E^r} \right] = \mathbb{E} \left[ \sum_{t=1}^T (r_{t,i} - r_i) \mathbf{1}_{\{x_{t,i}=1\}} \mathbf{1}_{E^r} \right]$ $\leq \mathbb{E} \left[ \sum_{t=1}^T \widetilde{O}\left( \frac{1}{\sqrt{N_t(i)}} \right) \mathbf{1}_{\{x_{t,i}=1\}} \mathbf{1}_{E^r} \right]$ $\leq \mathbb{E} \left[ \sum_{t=1}^T \widetilde{O}\left( \frac{1}{\sqrt{N_t(i)}} \right) \mathbf{1}_{\{x_{t,i}=1\}} \right] \leq \widetilde{O}(\sqrt{T}).$

**\[2\]:** 类似地，$\mathbb{E} \left[ \sum_{t=1}^T \sum_{j=1}^k (C_{t,ij} - C_{ij}) x_{t,j} \mathbf{1}_{E^c} \right] \leq \widetilde{O}(\sqrt{T})$，其中 $E^c = \bigcap_{t,i,j} \left\{ 0 \geq C_{t,ij} - C_{ij} \geq -\widetilde{O}\left( \frac{1}{\sqrt{N_t(j)}} \right) \right\}.$ $\square$
